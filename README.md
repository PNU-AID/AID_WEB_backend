## Commit Convention

> Type: subject
>
> Body
>
> Footer(이슈 트랙시 사용)

**Type**
- feat: 새로운 기능 추가
- fix: 버그 수정
- style: 코드 변경 없이 포맷만 변경한 경우
- refactor: 코드 리팩토링
- docs: 문서 수정
- rename: 코드변경 없이 파일 명 수정
- remove: 파일 삭제할 경우
- comment: 주석 추가
- test: 테스트 코드 추가
- chore: 빌드 관련 파일 수정, 패키지 추가 및 수정
